Dark Mode

To add dark mode to your site, add two CSS variables named --white and --black to your <body> tag, each with the opposite value. 

<body style="--white: black; --black: white;">
  <h1>
    Hi!
  </h1>
  <p>
    This page has dark mode.
  </p>
</body>
